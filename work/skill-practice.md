# Skill Practice — 7h/tuần (cam kết)

Mục tiêu: duy trì nhịp học/đào sâu kỹ thuật ổn định, không phụ thuộc mood.

## Lịch cố định mỗi tuần
- 2 phiên **90 phút**
- 2 phiên **45 phút**
- Tổng: **~5 giờ** (còn ~2 giờ buffer cho đọc/nghỉ/ôn)

> Chủ đề hiện tại (Feb 2026): **Distributed patterns**

## Cách dùng (nhanh)
- Trước mỗi phiên: viết 1 dòng mục tiêu (what output?)
- Sau mỗi phiên: ghi 3 dòng
  - Done:
  - Next:
  - Blockers:

## Checklist mẫu cho 1 phiên 90p (deep work)
- (10p) xác định 1 vấn đề/flow cụ thể
- (60p) implement/đọc code + ghi note
- (20p) chốt output: PR draft / note 1 trang / test case / diagram

## Checklist mẫu cho 1 phiên 45p (maintenance)
- (10p) đọc/nhặt 1 concept
- (25p) áp vào 1 ví dụ trong codebase
- (10p) note lại + tạo “next step”
